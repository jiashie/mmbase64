微信聊天消息自定义加密解密辅助工具
用法：
* 界面上选中复选框跳转到手机辅助服务，开启微密。
* 在输入框，输入wm:前缀，再输入想要加密的内容，然后长按输入框，会自动替换成mw:前缀的密文，然后再发送。
* 在消息列表，长按密文（mw:前缀)，会弹出toast，内容为解密后的明文。

默认密码bigBrotherIsWatchingYou。在微信群里事先约定好密码，群成员都设置为一致的密码方可解密。
